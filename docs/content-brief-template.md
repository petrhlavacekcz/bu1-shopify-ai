# Content Brief

**As of April 2026, the content brief is merged into the draft file's YAML frontmatter.** There is no longer a separate brief document — one article = one file in `drafts/blog/`.

See [`docs/blog-articles.md`](blog-articles.md) → **Draft File Format** for the canonical schema.

## Minimum brief fields (all live in draft frontmatter)

- `title`
- `primary_keyword`
- `secondary_keywords`
- `audience`
- `search_intent`
- `cluster` (1–5, see [`topic-cluster-map.md`](topic-cluster-map.md))
- `business_goal`
- `internal_links` (1–3 from `evergreen_pages` in config)
- `related_articles` (≥1, see Inter-Article Linking in blog-articles.md)
- `evidence_sources` (≥2 credible sources — cited inline, never as a Zdroje section)

## BU1-specific editorial rules

- Prefer goalkeeper, goalkeeper training, youth development, glove choice/care, and match-situation topics.
- Solve a real problem for player, parent, or coach.
- Avoid broad football culture unless there is a clear business reason.
- Link to evergreen pages before product collections.
