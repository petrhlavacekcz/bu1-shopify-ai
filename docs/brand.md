## Brand Voice

- **Tagline**: "Happy to help you play"
- **Tone**: Přátelský a profesionální. Krátké věty. Jasná pointa.
- **Address**: Vždy vykání (vy/vám/vaše), nikdy tykání
- **Sentences**: Max 15–20 slov. Jeden nápad na odstavec. Nejdůležitější info první.
- **CTA**: Max 2–4 slova. Jeden CTA na výstup. Specifický a uklidňující.
- **Forbidden**: "revoluční", "prémiový", "unikátní", vymyšlená tvrzení, umělé sociální důkazy, více CTA
- **Emoji**: Sociální sítě volně (1-3), web/B2B střídmě, vždy funkčně

## Content Quality Rules

- **Factual accuracy**: cite real sources (FIFA, UEFA, scientific studies) — no invented claims
- **Concrete data**: specific stats, names, events — no filler words or vague promises. Write "podle studie UEFA z roku 2023", not "mnoho expertů tvrdí". Write concrete reasons WHY, not "je důležité".
- **Storytelling**: based on real events and stories, never invented scenarios
- **AIDA structure**: Attention → Interest → Desire → Action
- **One CTA per section**, product/action before atmosphere
- **No sources section**: do NOT add a "Zdroje" / "Sources" section at the end of articles. Cite sources inline within the text naturally.
- **HTML tags**: h1-h3, p, ul, ol, li, a, strong, em, table
