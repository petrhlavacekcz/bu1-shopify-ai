# BU1 Topic Cluster Map

## Purpose

This map keeps the BU1 blog focused on areas where the brand has real authority and commercial relevance.

Each new article should belong to one cluster.

## Cluster 1: Goalkeeper Fundamentals

Audience:

- children
- dorost
- amateur goalkeepers
- parents

Core intent:

- learn the basics well
- improve technique safely

Priority topics:

- brankářská rozcvička
- postřeh a reakce
- postavení brankáře
- pádová technika
- jistota v rukou

Strong internal link targets:

- `https://bu1.cz/pages/poradna-jak-spravne-vybrat-brankarske-rukavice`
- `https://bu1.cz/pages/pece-o-rukavice`
- `https://bu1.cz/collections/brankarske-rukavice`

## Cluster 2: Match Situations

- střely po zemi

Strong internal link targets:

- related goalkeeper fundamentals articles

## Cluster 3: Goalkeeper Distribution And Modern Play

Audience:

- relevant match-situation and fundamentals articles

## Cluster 5: Youth Goalkeeper Development

Audience:

- rodiče
- trenéři
- děti
- dorost

Core intent:

- guide long-term development, not only isolated technique

Priority topics:

- tréninkový plán pro mladého brankáře
- nejčastější chyby mladých brankářů
- mentální odolnost po chybě
- jak trénovat doma

Strong internal link targets:

- fundamentals articles
- glove education evergreen pages where relevant

## Editorial Rules

- Every article must map to exactly one primary cluster.
- Articles may support one adjacent cluster, but should not try to cover three or four at once.
- Prefer evergreen goalkeeper education over broad football culture topics.
- Broad football history, transfer, and lifestyle topics should be secondary, not the main editorial engine.
